ggplot2¶

The ggplot2 library is part of a family of R packages called the tidyverse. See the ggplot2 documentation for a summary of all the functions in this library. See also the R Graphics Cookbook and the Data Visualizations chapter of R for Data Science for more explanation and examples.

library(ggplot2)

Note

If you try to execute a ggplot2 function before loading the library, you will get an error messages stating that R could not find the function.

# We will also be using dplr functions in the examples. 
library(dplyr)

Attaching package: ‘dplyr’

The following objects are masked from ‘package:stats’:

    filter, lag

The following objects are masked from ‘package:base’:

    intersect, setdiff, setequal, union

Set default plot dimensions¶

In some browsers, the plots may be too small or too big. You can control the size of the plots using the repr package to set the plot width and height as shown below.

# Resizes all plots to 5 in by 4 in
library(repr)
options(repr.plot.width=5, repr.plot.height=4)

Bar Charts¶

Bar Charts are used to show how quantities depend on a particular categorical variable. There are two ways to produce bar charts with ggplot. Use geom_bar() to visualize the number of observations in each category.

ggplot(DATA_FRAME, aes(x = COLUMN)) + geom_bar()

Use geom_col() to visualize how another numerical variable depends on the category.

ggplot(SUMMARY_TABLE, aes(x = COLUMN_1, y = COLUMN_2)) + geom_col()

Example: Number of chicks weighed on the first day¶

We can combine filter() and ggplot to create a bar chat that shows the number of chicks in each diet group that were weighed on the first day of the study.

# without piping
ChickWeight_start <- filter(ChickWeight, Time == 0)
ggplot(ChickWeight_start, aes(x=Diet)) + geom_bar()

# with piping
ChickWeight %>%
    filter(Time == 0) %>%
    ggplot(aes(x=Diet)) +
        geom_bar()

Example: Average weight on last day¶

ChickWeight %>%
    filter(Time == max(Time)) %>%
    group_by(Diet) %>%
    summarize(Avg_Weight = mean(weight)) %>%
    ggplot(aes(x = Diet, y = Avg_Weight)) + geom_col()

Style Tip

While using bar charts to represent averages is a common practice, other types of visualizations like box plots and histograms like do a much better job of representing how numerical values depend on categorical variables.

Histograms¶

Histograms are use to show the distribution of the values of a numeric variable.

ggplot(DATA_FRAME, aes(x = COLUMN)) + geom_histogram(bins = NUMBER_OF_BINS)

Example: Distribution of weights on the last day¶

# without piping
ChickWeight_last_day <- filter(ChickWeight, Time == max(Time))
ggplot(ChickWeight_last_day, aes(x = weight)) + geom_histogram(bins = 10)

# with piping
ChickWeight %>%
    filter(Time == max(Time)) %>%
    ggplot(aes(x = weight)) +
        geom_histogram(bins = 10)

Adding color allows us to distinguish between the different diets.

# with piping
ChickWeight %>%
    filter(Time == max(Time)) %>%
    ggplot(aes(x = weight, fill = Diet)) +
        geom_histogram(bins = 10, position = "dodge")

Scatter Plots¶

Scatter plots are used to show the relation ship between two numeric variables. Using color or shape we can also add information about categorical variables.

ggplot(DATA_FRAME, aes(x = COLUMN_1, y = COLUMN_2)) + geom_point()

# with piping
ChickWeight %>%
    filter(Diet == 4) %>%
    ggplot(aes(x = Time, y = weight)) +
        geom_point()

ggplot(ChickWeight, aes(x=Time, y=weight, color=Diet, shape=Diet)) + geom_point()

Box plots¶

Box plots are another way to show the distribution of a numeric variable. The bold horizontal line in the middle of the box indicates the location of the median. The top of the box is the cut off for the 75% quantile and the bottom is that for the 25% quantile. The top of the upper vertical whisker is the maximum value and the bottom of the lower whisker is the minimum value.

ggplot(aes(x = CATEGORIC_VARIABLE, y = NUMERIC_VARIABLE)) + geom_boxplot()

Example: Weights of chicks on last day¶

ChickWeight %>%
    filter(Time == max(Time)) %>%
    ggplot(aes(x = Diet, y = weight)) +
        geom_boxplot()
