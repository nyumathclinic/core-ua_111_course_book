Exercise 7: Percentage under 18¶

bronx_pop_density contains population data in the various ZIP codes in the Bronx in 2010. One column of this data set tells us the total population in each ZIP code; another column tells us the population under 18 in each ZIP code.

Suppose that we are interested in measuring how young each ZIP code is by determining percentage of residents under 18.

  • Load the dplyr package.

  • Create a new column called percentage_under18, which contains the percentage of each ZIP code’s population who are under 18.

  • Verify that the percentages are between 0 and 100.
