VectorsÂ¶

In R, vectors are ordered collections of entries where each entry has the same type. We use c() to combine multiple values into a single vector.

c(VALUE_1, VALUE_2, VALUE_3,...)

Vectors of consecutive integers can be constructed using the colon operator.

START_VALUE:END_VALUE

Type coercionÂ¶

The entires in a vector must all have the same data type. When applied to a vector, thetypeof() function returns the data type of the entries in that vector.

typeof(1:20)
'integer'
typeof(c('cat', 'dog', 'turtle', 'fish'))
'character'
typeof(c(1.75, 0.5, 100))
'double'
typeof(c(TRUE, FALSE, TRUE, TRUE))
'logical'

Warning

If vector entries are not of the same type, R will coerce them into being the same type.

typeof(1)
typeof('cat')
typeof(TRUE)

c('cat', 1, TRUE) # entries inputed with three different types
'double'
'character'
'logical'
  1. 'cat'
  2. '1'
  3. 'TRUE'
typeof(c(1, 'cat', TRUE)) # entries are ALL stored as charcters
'character'

R changed both the double and the logical variables to character variables. Notice the quote marks around 1 and TRUE in the output.

Computing with vectorsÂ¶

Arithmetic operations that combine a vector with a single number distributed to all entries in the vector.

c(1,2,3)*5 # Each entry is multiplied by five.
  1. 5
  2. 10
  3. 15

Arithmetic operations that combine multiple vectors are performed entry-by-entry.

c(1,2,3) - c(6,5,4) # Same output as c(1-6, 2-5, 3-4)
  1. -5
  2. -3
  3. -1

Combining vectorsÂ¶

The c() function can also be used to combine multiple vectors into a single vector.

c(VECTOR_1, VECTOR_2, VECTOR_3,...)

c(1:2,4:10) # All integers from 1 to 10 exept 3
  1. 1
  2. 2
  3. 4
  4. 5
  5. 6
  6. 7
  7. 8
  8. 9
  9. 10
cats <- c('Tiger', 'Boots', 'Fluffy')
dogs <- c('Spot', 'Rover', 'Buddy')
c(cats, dogs)
  1. 'Tiger'
  2. 'Boots'
  3. 'Fluffy'
  4. 'Spot'
  5. 'Rover'
  6. 'Buddy'
